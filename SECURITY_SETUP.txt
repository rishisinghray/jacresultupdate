# JAC Result Update – Security Setup Guide

## 1. Firebase Console – Firestore Rules
Go to: console.firebase.google.com
→ Project: ru-chat-ray
→ Firestore Database → Rules tab
→ Copy content from FIRESTORE_RULES.txt
→ Click "Publish"

## 2. Firebase Authentication
→ Authentication → Sign-in method
→ Enable: Email/Password ONLY
→ Disable: All other providers (Google, Facebook etc.)

## 3. Firebase Authentication – Users
→ Authentication → Users
→ Only ONE user should exist: jacresultupdate@gmail.com
→ Delete any other users
